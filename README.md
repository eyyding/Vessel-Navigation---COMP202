# Vessel-Navigation---COMP202
A text-based Python navigation game where a vessel races against a storm to reach a waypoint, featuring random wave impacts, distance tracking, and an interactive command menu.


## Features

- Interactive text-based navigation menu
- User-defined starting location and waypoint
- Great-circle distance calculation using the Haversine formula
- Step-by-step vessel movement toward a target
- Random wave events that alter vessel position
- Storm countdown that limits the number of moves
- Win and loss conditions based on distance and time
